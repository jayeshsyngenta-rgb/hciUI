SLIDE 2 - PROBLEM STATEMENT
Heading: "The Problem"
Content:
- 75% of college students report moderate-to-high stress during exams
- Existing study tools focus only on productivity, ignoring mental health
- Students lack accessible, integrated tools to manage study stress in real-time
- Stress leads to poor focus, burnout, and lower academic performance
Visual suggestion: Stress statistics infographic or student stress illustration

SLIDE 3 - OUR SOLUTION
Heading: "Our Solution"
Content:
- An all-in-one web app that combines study management with stress relief
- Personalized recommendations based on self-reported stress levels
- Evidence-based techniques: Pomodoro, breathing exercises, ambient sounds
- AI-powered study companion for emotional support and study tips
Visual suggestion: App screenshot or hero section mockup

SLIDE 4 - HCI PRINCIPLES APPLIED
Heading: "HCI Design Principles"
Content:
- User-Centered Design: Stress level drives the entire experience
- Feedback & Visibility: Progress bars, mood tracking, achievement badges
- Flexibility: Dark/light mode, customizable timers, multiple sound options
- Error Prevention: Guided 4-step session wizard eliminates confusion
- Aesthetic & Minimalist Design: Clean UI with Framer Motion animations
Visual suggestion: Side-by-side UI screenshots showing principles

SLIDE 5 - KEY FEATURES (Part 1 - Study Tools)
Heading: "Features: Study Management"
Content (with icons):
- Pomodoro Timer: 4-cycle work/break timer with auto-progression
- Study Planner: Subject & topic management with checkboxes
- AI Study Companion: Gemini-powered chatbot for study help & emotional support
- Focus Rating: Post-session 5-star self-assessment
- Study Journal: Guided reflection prompts after each session
Visual suggestion: Feature grid with app screenshots

SLIDE 6 - KEY FEATURES (Part 2 - Wellness)
Heading: "Features: Stress Relief & Wellness"
Content (with icons):
- Breathing Exercises: 3 presets (Box, 4-7-8, Calm) with animated guide
- Ambient Sounds: Rain, Lo-Fi, Nature with volume control
- Mood Tracker: Before/after session mood check-ins with 14-day trend chart
- Virtual Garden: Plants grow as you complete study sessions (gamification)
- Achievement Badges: 12 unlockable badges, XP system, 5 levels
Visual suggestion: Feature grid with wellness UI screenshots

SLIDE 7 - SYSTEM ARCHITECTURE
Heading: "System Architecture"
Content:
- Frontend: React.js + Vite + shadcn/ui + Framer Motion + Tailwind CSS
- Backend: Flask (Python) + Gunicorn
- AI Integration: Google Gemini API (generative AI for chat)
- Data Storage: LocalStorage (client-side, privacy-first)
- Hosting: Render.com (backend Web Service + frontend Static Site)
- Design Pattern: RESTful API, component-based UI
Visual suggestion: Architecture diagram showing Frontend <-> Backend <-> Gemini API

SLIDE 9 - RESULTS & IMPACT
Heading: "Impact & Evaluation"
Content:
- 20+ interactive features in a single cohesive application
- Addresses both productivity AND mental wellness simultaneously
- Gamification (garden, badges, XP) increases engagement and return usage
- Mood tracking enables students to see correlation between study habits and emotional state
- PDF export feature allows students to share progress with counselors/advisors
